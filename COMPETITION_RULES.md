# CAFA 6 Competition Rules Summary

## Basic Info

| Item | Value |
|------|-------|
| Competition | CAFA 6 Protein Function Prediction |
| Sponsor | Iowa State University |
| Prize Pool | $50,000 (1st: $15k, 2nd: $10k, 3rd: $8k, 4th: $7k, 5th-6th: $5k) |
| License | Open Source (MIT) |

## Submission Rules

| Rule | Limit |
|------|-------|
| Daily submissions | 5 per day |
| Final submissions | 2 selectable |
| Team size | Max 5 members |

## External Data: ALLOWED

External data is permitted under these conditions:

1. **Publicly available** - Accessible to all participants at no cost
2. **Reasonableness standard** - No excessive cost barriers

Examples of OK external data:
- GOA (Gene Ontology Annotations)
- UniProt
- InterPro
- Public pretrained models (ESM-2, ProtT5, etc.)

## Code Sharing Rules

| Type | Allowed? | Notes |
|------|----------|-------|
| Private sharing | NO | Cannot share code/data outside your team |
| Public sharing | YES | Only via Kaggle forums/notebooks |
| Open source libraries | YES | Must use OSI-approved license |

## Winner's Obligations

If you win, you must:

1. **Open Source License (MIT)** - Release winning solution code
2. **Version Control** - Track code via GitHub/GitLab/Bitbucket
3. **Archive** - Archive release on Zenodo or similar platform
4. **Documentation** - Provide reproducible description including:
   - Architecture
   - Preprocessing
   - Loss function
   - Training details
   - Hyperparameters
   - Computational environment

## Prohibited Actions

- Multiple Kaggle accounts
- Hand labeling or human prediction of test data
- Private code/data sharing between teams
- Tampering with submission process

Check the competition timeline page for:
- Entry Deadline
- Team Merger Deadline
- Final Submission Deadline
